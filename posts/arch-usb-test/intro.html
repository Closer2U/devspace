Introduction

This is the intro page for the Arch USB guide.

Why Arch on USB?

Running Arch Linux from a USB drive gives you a portable, persistent system you can boot on any compatible machine.

Prerequisites

You will need:

  • A USB drive (8GB+)
  • A working Linux or macOS system
  • The
    # dd
    command or a tool like Etcher
